What is OKDP?

OKDP (Open Kubernetes Data Platform) is a comprehensive data management services platform composed of containerized open source software and products, running on Kubernetes infrastructure.

OKDP addresses the complete data lifecycle: collection, storage, processing, analysis, and data exposition. It's modular by design‚Äîdeploy all components or select only what you need.
